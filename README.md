   ```bash

   ```bash

   npm install

   ```bash

   npm start

   ```bash
   cd ../server
   npm start
   ```

7. Open your web browser and navigate to `http://localhost:3000` to view the client-side application.

8. The server will be running at `http://localhost:your-server-port`.
